# Announcement Images

This folder contains images for the Featured Announcements system.

## Required Images (for mock data)

Please add the following images to this folder:

1. `welcome-back.jpg` - Welcome back to school image
2. `science-fair.jpg` - Science fair/laboratory image
3. `parent-conference.jpg` - Parent-teacher meeting image
4. `reading-month.jpg` - Books/reading/library image
5. `sports-day.jpg` - Sports/athletics image
6. `arts-week.jpg` - Arts and culture image
7. `green-initiative.jpg` - Environmental/nature image
8. `computer-lab.jpg` - Technology/computers image
9. `feeding-program.jpg` - Community/food program image

## Image Specifications

- **Format**: JPG, PNG, or WebP
- **Dimensions**: Minimum 800x600px (recommended: 1200x800px)
- **Aspect Ratio**: 3:2 or 16:9
- **File Size**: Under 500KB (optimized for web)
- **Quality**: High quality, well-lit, professional photos

## Temporary Solution

Until you add real images, you can:
1. Use placeholder images from https://picsum.photos/1200/800
2. Use stock photos from free sources like Unsplash or Pexels
3. Take photos of your school events and activities

## Adding New Announcements

When adding new announcements through the admin interface (Phase 2), images will be automatically uploaded to this folder.
